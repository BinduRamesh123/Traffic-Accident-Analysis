## Project Overview
This project analyzes traffic accident data to identify patterns and risk factors contributing to road accidents. 

The project includes:
- Data Cleaning & Exploratory Data Analysis (Python)
- Data Visualization (Matplotlib & Seaborn)
- Interactive Excel Dashboard


## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Excel (Pivot Tables, Charts, Slicers)
- Google Colab

---

## Key Insights
- Accident cases show moderate class imbalance.
- Alcohol consumption increases accident likelihood.
- Higher speed limits are associated with increased accident risk.
- Weather conditions like rain and snow show higher accident proportions.
